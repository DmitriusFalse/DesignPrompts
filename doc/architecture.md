# Архитектура

## Обзор

Монолитное десктопное приложение: Go─бэкенд встраивает статические файлы фронтенда и запускает HTTP─сервер на `127.0.0.1`. Пользователь взаимодействует через встроенное окно WebView2.

## Компоненты

### Бэкенд (Go)
┌──────────────────────────────────────────────────────────────────────────────┐
│ Пакет      │ Назначение                                                      │
│────────────│─────────────────────────────────────────────────────────────────│
│ `main`     │ Точка входа, embed `version.txt`, инициализация, запуск сервера │
│ `config`   │ Загрузка/сохранение `config.json`, дефолты, resolve путей       │
│ `database` │ SQLite: миграции, репозиторий (CRUD), модели данных             │
│ `handler`  │ HTTP─роуты, хендлеры, статика, API                              │
│ `sync`     │ Сканирование паков, парсинг CSV/TXT, hasher, синхронизация с БД │
│ `logger`   │ Логирование в файл + stderr, ротация по дате                    │
└──────────────────────────────────────────────────────────────────────────────┘
### Фронтенд (SPA)

│ Файл │ Назначение │
│──────│────────────│
│ `index.html` │ Основная страница, Alpine.js корневой компонент (+ ComfyUI generation UI) │
│ `settings.html` │ Страница настроек (+ ComfyUI секция) │
│ `js/app.js` │ Логика: чипы, drag─and─drop, промпты, пресеты, автосохранение, ComfyUI (WS, generate) │
│ `styles.css` │ Утилитарный CSS (Tailwind─like), светлая/тёмная тема (+ ComfyUI: tabs, progress, gen controls) │

│ `presets.json` │ Готовые пресеты промптов │
│ `sw.js` │ Service Worker: cache─first для статики, network─first для API │
│ `manifest.json` │ PWA─манифест │

## Стартовая последовательность

1. `main()` определяет путь к `config.json` (рядом с exe)
2. Загружает конфиг (создаёт дефолтный, если нет)
3. Инициализирует логгер
4. Открывает SQLite (WAL mode, foreign keys)
5. Запускает первоначальную синхронизацию паков
6. Регистрирует HTTP─роуты + `/api/version`
7. Запускает HTTP─сервер в горутине
8. Запускает окно WebView2 (блокирующий вызов)
9. После закрытия окна — graceful shutdown

## Ключевые решения

─ **Single binary**: все статические файлы встроены через `//go:embed`
─ **Локальный сервер**: `127.0.0.1`, никакой внешней сети
─ **WAL mode**: конкурентное чтение без блокировок
─ **SHA─256 хеши**: инкрементальная синхронизация файлов
─ **info.pack**: метаданные пака не перезаписываются при повторном сканировании
─ **`─H=windowsgui`**: без консольного окна
─ **Alpine.js SPA**: без сборки, CDN, реактивность через атрибуты
─ **CSS custom properties**: светлая/тёмная тема без перезагрузки
─ **gorilla/websocket**: прокси WebSocket от браузера к ComfyUI (решает CORS 403)
─ **ComfyUI SaveImage**: воркфлоу сохраняет с простым префиксом (`Danbooru`), наш бэкенд копирует результат в `cfg.SavePath` через `/api/comfy/save-image`
─ **Макроподстановка**: два прохода — `"%KEY%"` (JSON-escaped строка) и `%KEY%` (сырое значение)
─ **Seed**: `int64`, с возможностью фиксации (чекбокс «Fix») — при выключенном фиксе генерируется каждый раз
